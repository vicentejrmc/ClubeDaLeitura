# ClubeDaLeitura

- **Projeto desenvolvido durante o curso Full-Stack da Academia do Programador 2025**

- O Sistema consiste no gerenciamento de um 'clube' onde são compartilhadas revistas, Hqs, Quadrinhos ou livros.

O Sistema organiza os intens em caixas, tendo cada uma delas uma cor, titulo e tempo limite de dias para emprestimo.

- O Projeto foi conceituado usando Programação Orientada a Objetos na linguagem C# funcinando no modo Console.App 

## Fucnionalidades dessa Versão data 29/04/25

- Cadastro  -  (Amigo, Caixa, Revista Emprestimo)
- Editar  - (Amigo, Caixa, Revista Emprestimo)
- Visualização - (Amigo, Caixa, Revista Emprestimo)
- Exclusão   - (Amigo, Caixa, Revista Emprestimo)

- Devlolução de (Emprestimo)

- **istema conta com Validações para**:
