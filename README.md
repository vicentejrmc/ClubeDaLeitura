# ClubeDaLeitura2025

- **Projeto desenvolvido durante o curso Full-Stack da Academia do Programador 2025**

Sistema de gestão de equipamentos: O projeto foi desenvolvido com o uso da linguagem C#,
e sistema funciona dentro do sistema ConsoleApp do DotNet.

Consiste no gerenciamento de um 'clube' onde são compartilhadas revistas, Hqs, Quadrinhos ou livros.

O Objetico é organizar os intens(Revistas Hqs etc.) em caixas, tendo cada uma delas uma cor, titulo e tempo limite de dias para emprestimo.
O Sistema possibilida a criação, edição, exclusão e visualizaçao das Entidades criadas.


## Funcionalidades de Menu.

- Cadastro  -  (Amigo, Caixa, Revista Emprestimo)
- Editar  - (Amigo, Caixa, Revista Emprestimo)
- Visualização - (Amigo, Caixa, Revista Emprestimo)
- Exclusão   - (Amigo, Caixa, Revista Emprestimo)
- Devlolução de (Emprestimo)

## Utilização:

1. Clone o repositorio ou baixe o codigo fonte.
2. Abra o Terminal ou prompt de Comando e navegue até a pasta raiz.
3. Utilize o camando abaixo para restaurar as dependências do projeto.

```
dotnet restore
```

4. Compile a solução utilizando o comando:
```
 dotnet build --configuration Release
```

5.Para executar o projeto compilando em tempo real
```
dotnet run --project ClubeDaLeitura.ConsoleApp
```

6. Para executar o arquivo compilado, navegue até a pasta 
/ClubeDaLeitura.ConsoleApp/bin/Release/net8.0/ e execute o arquivo: GestaoDeEquipamentos.ConsoleApp.exe
